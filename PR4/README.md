## Архітектура моделей

### LSTM (Long Short-Term Memory)

Нейронна мережа побудована на основі PyTorch з такими параметрами:

| Параметр | Значення |
|---|---|
| Довжина вхідної послідовності | 24 години |
| Кількість LSTM шарів | 2 |
| Розмір прихованого шару | 64 нейрони |
| Dropout | 0.2 |
| Оптимізатор | Adam (lr=0.001) |
| Функція втрат | MSELoss |
| Кількість епох | 20 |
| Розмір батчу | 64 |

### ARIMA

Класична статистична модель авторегресії з ковзним середнім та інтегруванням. Параметри: `order=(2, 1, 2)` — 2 авторегресійних члени, 1 диференціювання, 2 члени ковзного середнього.

### Prophet

Модель від Meta (Facebook) для прогнозування часових рядів з урахуванням сезонності: добової, тижневої та річної.

### Крок 4 — Метрики точності

Для кожної моделі розраховуються три метрики:

## Висновок

Порівняння трьох підходів до прогнозування часових рядів показало відмінності між нейронними мережами та класичними статистичними методами. LSTM здатна вловлювати складні нелінійні залежності у погодинних даних завдяки механізму пам'яті. ARIMA добре працює на стаціонарних рядах, але потребує диференціювання для усунення тренду. Prophet автоматично враховує сезонність різних рівнів і є зручним у використанні без тонкого налаштування. Метрики MAE, RMSE та MAPE дозволяють кількісно порівняти точність кожного методу на реальних даних.
